Frequently Asked Questions

How do I create an account?

Select create account tab from the home page. From there you should be able to log in!

How do I create a new list?

Make sure you are logged in. From there select the Create New List button. You will be redirected to a new page where you can create your list.

How do I add a new item to a list?

View a list. From there select the Create New Item button. You will be redirected to a new page where you can create your item.
